## 简介

工具库：php客户端输入/输出、字符串处理(手机/名字/身份证打码、字符串提取等)、字符串校验(邮件、手机、身份证校验等)、配置加载


## composer install

```
{
  "repositories": [
    {
      "url": "https://github.com/daniel1988/onelib.git",
      "type": "git"
    }
  ],
  "require": {
        "php": ">=5.4.0",
        "OneLib" : "dev-master"
    },
  "autoload": {
        "psr-0": {
            "OneLib": "src/"
        }
    }
}
```
